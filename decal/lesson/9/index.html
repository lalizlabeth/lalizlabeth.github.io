What is User Interface Design?

There are many definitions of user interface (UI) design. UI is broadly defined as the design of mobile applications, web pages, televisions, remote controls, etc. Let's say that the activity is eating a bowl of cereal. The user interface for this task would be your spoon.

Virtually everything is a user interface: Telebears, Facebook, Instgram, etc.

Some Basic UI Principles

Highlight important changes

Notifications are a good example of change. After clicking on a notifications icon, the notification count is reset to 0 and the red dot disappears!

Customization

The ability to personalize an experience increases the overall satisfaction of any user. Twitter, for example, allows their users to choose a personal background for their page.

Feedback

It's important to give a user feedback when they use an application. Forms frequently give feedback to their users. For example, when you type a password incorrectly into a login form, the field will usually turn red and give you an error.

Helpers

Icons aren't always self-explanatory. Having helper text or tooltips will give the user more security when they first use an application.

Patterns

Notice design patterns! If you look at mail applications, they're generally set up in a directory style. Straying from this design pattern is dangerous because users are familiar with it.

Consistency

Make sure your desgin is consistent across different pages of your application. Examples of this include: navigation, icons, typography. etc.
