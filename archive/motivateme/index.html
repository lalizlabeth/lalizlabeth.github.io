Web Development

motivateme

Summary

My friend and I created motivateme at a hackathon hosted by Hackers@Berkeley. The application generates a random personalized motivational quote. We created this application to motivate ourselves to do a hack.

Technologies Used

  • Flask
  • Javascript
  • HTML/CSS

    • The User Interface

    The UI is inspired by Microsoft's "Metro" design. We wanted to eliminate the need for the user to use his/her mouse. All the controls are mapped to the keyboard and flow nicely from screen to screen. Check out the live site here.
